# 贡献者行为准则

## 我们的承诺

为了营造开放且受欢迎的环境，我们承诺：
- 将参与视为无论年龄、体型、外观、种族、性别、性别认同、性取向、残疾、宗教、国籍或社会经济地位的人的包容机会

## 我们的标准

有助于创建积极环境的行为包括：
- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同情

不可接受的行为包括：
- 使用性别化语言或意象
- 人身攻击、贬低评论或政治攻击
- 骚扰、跟踪或恐吓
- 未经许可发布他人的私人信息
- 违反合理隐私和安全预期的其他行为

## 执行

不可接受行为的实例可以通过联系项目团队报告。所有投诉将被审查和调查。

项目维护者有责任澄清可接受行为的标准，并应对不可接受行为采取适当的纠正行动。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)
